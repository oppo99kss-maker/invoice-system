# نظام الفوترة المرن (نظام إدارة الفواتير وعروض الأسعار)

نظام ويب محلي وبسيط لإصدار وإدارة الفواتير وعروض الأسعار وعقود الصيانة، مع دعم كامل للغة العربية والإنجليزية والطباعة كـ PDF وتصميم عصري يدعم الوضعين الداكن والفاتح.

## حسابات تسجيل الدخول الافتراضية

تم تأمين النظام بشاشة دخول وحسابات مستقلة لكل مستخدم (عزل تام للبيانات):

1. **المستخدم الرئيسي (t)**:
   - اسم المستخدم: `t`
   - كلمة المرور: `t12345`
   - *ملاحظة*: يرث هذا الحساب تلقائياً كافة البيانات المخزنة مسبقاً على هذا الجهاز.

2. **المستخدم الثاني**:
   - اسم المستخدم: `user2`
   - كلمة المرور: `user2pass`

3. **المستخدم الثالث**:
   - اسم المستخدم: `user3`
   - كلمة المرور: `user3pass`

4. **المستخدم الرابع**:
   - اسم المستخدم: `user4`
   - كلمة المرور: `user4pass`

5. **المستخدم الخامس**:
   - اسم المستخدم: `user5`
   - كلمة المرور: `user5pass`

## طريقة التشغيل

1. افتح ملف `index.html` في أي متصفح ويب (مثل Google Chrome أو Microsoft Edge).
2. قم بتسجيل الدخول باستخدام أحد الحسابات أعلاه.
3. يمكنك تبديل الوضع الداكن/الفاتح أو تعديل الإعدادات والبدء في إنشاء الفواتير.
